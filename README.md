# 🚀 21 Days of Problem Solving with JavaScript

A self-imposed 21-day challenge to sharpen JavaScript problem-solving skills — from basic number logic to trees and recursion. Every day covers a new topic with **8–10 hands-on practice questions**.

---

## 🎯 Goal

To build a strong foundation in JavaScript logic, data structures, and algorithms through consistent daily practice — one topic at a time, no skipping.

This is a personal challenge to practice problem solving every single day for 21 days straight. The topics are carefully ordered — starting from the basics and gradually moving toward more complex concepts like recursion and trees.

## 📌 How to Run
```bash

git clone https://github.com/YOUR_USERNAME/21-days-of-problem-solving-with-javascript.git

# Navigate into any day's folder
cd Day-01-Number-Problems

# Run a solution file
node solution1.js
```
